As an Intern to Part-Time Full Stack Software Engineer at Ecumen, I played a crucial role in the development of a SaaS platform for long-term healthcare facilities. My responsibilities included designing and implementing new UI and backend features, ensuring usability and accessibility for the end-users, and utilizing SDLC agile methodology for monthly releases, including bug fixes, UI enhancements, and new features.

One of the most significant contributions I made was developing a map builder tool that simplified and reduced the time needed to program facility maps for visually tracking infections. Previously, the process of manually placing HTML div elements on the map was time-consuming and prone to errors. However, with my map builder tool, users could directly draw rooms on the map using the mouse, reducing map programming time from hours to just a few minutes.

The map builder tool was integrated with a database to update room names for easy infection tracking in each facility. This project not only saved time and increased accuracy but also provided a user-friendly and intuitive interface for the end-users.

Overall, my role at Ecumen allowed me to gain valuable experience in full-stack development, agile methodology, and project management. My contributions, particularly in the map builder project, streamlined the user experience, improving the accuracy and efficiency of infection tracking in long-term healthcare facilities.
